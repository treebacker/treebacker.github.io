IDC脚本学习

基础

  • IDC变量
    • 支持整数、字符串、浮点型
    • 字符串是IDC的本地类型
    • 局部声明变量auto（可以有初始值）；全局声明变量global（不可以有初始值）
    • IDC不支持数组
  • IDC表达式

    • 几乎支持C的所有算术和逻辑运算

    • 所有整数操作数所为有符号的值进行处理；（逻辑右移，手动处理）

    • 对于字符串

      • 无需strcat，strcpy，strdup等函数
      • 字符串的赋值操作即会复制
      • 字符串的相加操作即会将两个字符串拼接
      • 支持类似python的切片处理
  • IDC语句

    • 基本和C语言一样

    • 例外

      • IDC不支持switch复合语句

      • 在for结构中，IDC不支持复合赋值运算符

  • IDC函数

    • 函数定义以关键字static引入，参数只需要名，无需类型

    • 函数传参的方式由调用时决定（by value or by reference

    • 返回值根据执行路径不同而有差异

    • 不return时，默认返回0

  • IDC程序

    • 包含idc.idc文件，无参的main函数

  • IDC永久数据存储

    • 虽然IDC不支持一般意义的数组，却支持全局永久数据块

  • 关联IDC脚本与热键

    • 在/idc/ida.idc文件中main函数中添加AddHotkey的调用
