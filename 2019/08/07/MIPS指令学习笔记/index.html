MIPS学习笔记

寄存器

  • 有32个通用寄存器，三个特殊寄存器。

  • 字节序

    • 大端序、小端序；file指令查看(MSB 或者 LSB)

  • 寻址方式

    • 寄存器寻址 多用于load/store两种
    • PC寻址 多用于转移指令
    • 立即数寻址 多用于寻找变量

  • MIPS指令

    • 固定4字节长度

    • 内存中的数据访问必须严格对齐（4byte）

    • 流水线效应

      • 下指令的，strchr函数的参数来自$s0而不是$s2

    • 指令格式

    • 汇编常用指令

      • load/store指令

      • 算术指令

        • 算术指控得所有操作都是寄存器，不能是RAM地址或间接寻址

        • 操作数大小都是word（4byte）

        • 类比较指令

          • 虽然MIPS没有标志寄存器，但有SLT系列指令，通过比较设置某个寄存器后，与分支跳指令联合使用。

        • SYSCALL

          • 产生一个软化中断，实现系统调用；系统调用号存放在$v0中，参数在$a0~$a3中；

            返回值在$v0中，如果出错，在$a3中返回错误号；在编写shellcode时，用到该指令机制

MIPS程序经IDA和QEMU远程调试

  • 本质上，qemu内置gdbserver，开放端口，在端口监听，可以使用gdb，ida等连接调试
  • 运行时调试
    • 类似于在windows主机调试ELF文件，这样会有程序的符号表信息
    • 便于下断点，跟踪错误
  • 附加调试，类似与gdb的进程附加调试，没有符号信息

MIPS——shellcode开发

  • shellcode开发一般步骤
    • 用C语言开发shellcode
    • strace跟踪程序运行，得到所有调用的系统调用号
    • 用汇编语言模拟实现
    • 在gdb中调试，提取shellcode
    • 精简、去坏
